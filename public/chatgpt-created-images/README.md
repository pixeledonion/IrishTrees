# ChatGPT-Created Images

This folder is for images generated directly by ChatGPT in the shared conversation.

The downloaded files have been renamed with descriptive, reusable filenames. See `downloaded-images.json` for the original downloaded UUID names, current filenames, and short descriptions — that file is the authoritative record of what is in this folder.

In the Vite app, use paths like:

```text
/chatgpt-created-images/caledonian-pinewood-grove-7x10m-layout-plan.png
```
