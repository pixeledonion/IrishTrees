# ChatGPT Shared Chat Images

These images were downloaded from the shared ChatGPT conversation used for project context:

https://chatgpt.com/share/6a512fc0-cb4c-83eb-a9f9-eb7b646c6b64

The files are reference images for the IrishTrees project. `manifest.json` maps each local filename back to its original OpenAI image URL.

This folder has been curated down to the 104 images relevant to the Donegal pinewood-grove concept (Scots pine, birch, rowan, heather, ferns, sedges, willow catkins, deadwood, wildlife pond, and the birds/insects the grove supports). Off-brief images (retailer collages, product nest boxes, barren upland slopes) and near-duplicate frames were removed, so the numbering has gaps. `manifest.json` is the authoritative list of what remains.

Use paths like this inside the Vite app:

```text
/chatgpt-images/chatgpt-image-001.jpg
```
