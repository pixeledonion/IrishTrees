# Image Briefs For IrishTrees

These prompts are for GPT/Codex image generation. Save final project images into:

```text
public/generated-images/
```

Avoid relying on generated images for dense text. For infographics, generate the visual background or clean illustration, then let Claude build labels/text in React/HTML for accessibility.

## Visual Style

- Naturalistic, Irish Atlantic ecology.
- Professional educational tone.
- Soft overcast light, wet moss, native plants, real habitat texture.
- Avoid fantasy forests, oversized trees, tropical plants, people, buildings, signs, labels and fake UI text.

## Needed Images

### 1. Timeline Hero: Ireland Through Time

Create a wide 16:9 cinematic image showing a subtle transition from early post-glacial open ground to pine-birch woodland, oak-hazel woodland, bog/heath, and modern rewilding garden. It should feel like one continuous Irish landscape through time, not a collage. No text.

### 2. Pine-Birch-Hazel Wildwood

Create a naturalistic early Holocene Irish pine-birch-hazel woodland. Scots pine, birch, hazel, rowan, heather, bilberry, mossy stones, a wet hollow, deadwood, and soft Atlantic light. No text.

### 3. Atlantic Oakwood / Temperate Rainforest

Create a humid western Irish oakwood with sessile oak, hazel, holly, rowan, ferns, mosses, liverwort-rich rocks, lichens, and damp woodland floor. No text.

### 4. Raised Bog / Blanket Bog Habitat

Create an Irish peatland landscape with Sphagnum moss, bog cotton, heather, bog pools, sundew, open sky, and low western light. No text.

### 5. Garden Rewilding Before / After

Create a split but seamless 16:9 educational image: left side short lawn and tidy garden, right side rewilded native garden with pond, hedgerow, meadow strip, deadwood, native tree, bees and birds. No text.

### 6. Wildlife Pond Close View

Create a fish-free garden wildlife pond in Ireland with sedges, rushes, mossy stones, damselflies, frog, pond beetles, and reflections of birch and rowan. No text.

### 7. Student Learning Illustration

Create a clean educational illustration showing pollen, peat core, tree rings, fossil seeds, and a small Irish landscape behind them. Leave space for labels to be added later in HTML. No text.
